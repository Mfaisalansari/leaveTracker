Supabase Sync

Connect to Supabase

Stores leave data in a free cloud database. Everyone on the team sees the same data in real time — no personal accounts, no file sharing needed.

One-time setup (~10 min) — done once by team admin
  1. Go to supabase.com → New Project (free)
  2. Go to SQL Editor and run this query:
    create table leaves (
      id text primary key,
      name text, type text,
      start_date text, end_date text,
      reason text,
      created_at timestamptz default now()
    );
    alter table leaves enable row level security;
    create policy "public_all" on leaves for all using (true) with check (true);
  3. Go to Project Settings → API
  4. Copy Project URL and anon public key → paste below
This is the anon/public key — safe to share with the team.
